# Prompt — Testimonial / Social Proof Video

## Cible

Vidéo dynamique (30-45s) présentant des résultats chiffrés ou un témoignage client.

## Usage

- Section "Results" de la landing page
- LinkedIn / social media
- Email nurturing (closing argument)

## Template de prompt

```
Create a [DURATION]-second testimonial/results video.

Brand: Marin (accent #49C5B6, dark #015048, bg #FAFAFA, text #000000).
Font: Inter. Style: Data-driven motion graphics.

Scene structure:
1. [INTRO — 5s]: "[CLIENT TYPE] achieved these results with Marin"
2. [METRICS — 20s]: Animated numbers with labels
   - Metric 1: [NUMBER] [UNIT] — "[LABEL]"
   - Metric 2: [NUMBER] [UNIT] — "[LABEL]"
   - Metric 3: [NUMBER] [UNIT] — "[LABEL]"
3. [QUOTE — 10s]: "[CLIENT QUOTE OR TESTIMONIAL TEXT]"
4. [CLOSE — 5s]: "Your results start here" + CTA

Animation style: Numbers count up with spring bounce.
Bar charts draw from left to right.
Quote fades in with a subtle slide.
```

## Exemple — Résultats grossiste

```
Create a 40-second results video.

Brand: Marin (accent #49C5B6, dark #015048, bg #FAFAFA, text #000000).
Font: Inter. Style: Data-driven motion graphics.

Scene structure:
1. INTRO (5s): "A Lyon-based wholesaler reached their goal in 60 days"
2. METRICS (20s):
   - "12" — "deals closed" (with euro icon)
   - "€24,500" — "revenue generated" (line chart animating up)
   - "8.2%" — "conversion rate" (percent circle filling)
3. QUOTE (10s): "Marin replaced our entire cold calling team. Best decision we made."
4. CLOSE (5s): "Your turn." + CTA button "Book a call"

Animation: Numbers count up with spring bounce.
Line chart draws from left to right.
Percent circle fills with spring.
Quote text types in character by character.
```

## Exemple — Garantie

```
Create a 25-second results video focused on the guarantee.

Brand: Marin (accent #49C5B6, dark #015048, bg #FAFAFA, text #000000).
Font: Inter. Style: Bold, confident motion graphics.

Scene:
1. Large number "10" animates up from 0 with a strong bounce
2. Text appears below: "SALES GUARANTEED"
3. Sub-text: "90 days. Or we continue until you get them."
4. A shield icon appears around the number
5. Fade to CTA: "No risk. Just results."

Animation: Number counts up with heavy spring bounce.
Text fades in with slight letter-spacing expansion.
Shield icon grows from center with a rotation.
```
