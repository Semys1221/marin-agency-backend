# Libraries and Templates

## App Templates

Scaffold a working app with Shopify CLI:

```bash
shopify app init --flavor <template>
```

| Template | Stack |
|----------|-------|
| [React Router template](https://github.com/Shopify/shopify-app-template-react-router) | Recommended for most apps |
| [Extension-only template](https://github.com/Shopify/shopify-app-template-extension-only) | No developer-hosted backend |
| [Express template](https://github.com/Shopify/shopify-app-template-node) | Express, Node.js, React |
| [Ruby template](https://github.com/Shopify/shopify-app-template-ruby) | Ruby on Rails |
| [PHP template](https://github.com/Shopify/shopify-app-template-php) | PHP with Laravel |

## Official API Libraries

| Library | Language | Description |
|---------|----------|-------------|
| [shopify-api-ruby](https://github.com/Shopify/shopify-api-ruby) | Ruby | Admin API, OAuth, webhooks, REST + GraphQL |
| [shopify_app](https://github.com/Shopify/shopify_app) | Ruby/Rails | Rails engine with scaffolding |
| [shopify-api](https://github.com/Shopify/shopify-app-js/tree/main/packages/apps/shopify-api) | Node.js | Storefront + Admin APIs |
| [admin-api-client](https://github.com/Shopify/shopify-app-js/tree/main/packages/api-clients/admin-api-client) | Node.js | Lightweight Admin API client |
