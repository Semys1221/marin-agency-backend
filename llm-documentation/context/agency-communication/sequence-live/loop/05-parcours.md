# Parcours — loop.so

**Déclencheur :** J+15 après onboarding (quand la séquence onboarding est terminée)
**Type :** Nurturing / rétention
**Système :** loop.so
**Durée :** 90 jours

Un email par semaine, identique, qui fait le point sur l'avancée du client. Le contenu est générique pour fonctionner chaque semaine — il s'adapte via les variables dynamiques.

---

## Step 1 à 13 — Semaine 1 à 13 (un email toutes les 1 semaine)

**Sujet :** {{prenom}}, votre point parcours de la semaine ✨

```
Bonjour {{prenom}},

Comment s'est passée votre semaine chez {{entreprise}} ?

C'est le moment du point parcours — chaque semaine, on regarde où vous en êtes :

✅ Qu'est-ce qui a bien fonctionné cette semaine ?
🔄 Qu'est-ce qu'on peut améliorer ?
🎯 Prochaine étape : on continue sur la lancée

Vous commencez à voir les résultats arriver, et c'est normal — la régularité paie.

Si vous avez le moindre doute, une question, ou si vous voulez qu'on ajuste quelque chose, répondez simplement à cet email. Je suis là pour vous accompagner jusqu'au bout.

On continue ensemble 💪

{{gmail_signature}}
```

---

## Notes

- La séquence est identique chaque semaine car elle reflète le parcours global, pas une étape spécifique
- Variables dynamiques disponibles : `{{prenom}}`, `{{entreprise}}`, `{{gmail_signature}}`
- Si personnalisation supplémentaire souhaitée : `{{appels_semaine}}`, `{{ventes_semaine}}`, `{{jours_restants}}`
- Condition d'arrêt : fin du contrat (90 jours), résiliation, ou objectif atteint
