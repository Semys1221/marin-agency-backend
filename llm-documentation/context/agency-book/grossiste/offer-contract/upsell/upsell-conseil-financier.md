# Upsell Conseil Financier

- **Prix :** 800 €
- **Cible :** Client en fin de parcours 90 jours (J60+)

## Contexte

Après 60+ jours d'accompagnement, le client a généré des ventes et perçoit mieux les flux financiers de son activité. L'upsell Conseil Financier l'aide à structurer, optimiser et pérenniser sa trésorerie.

## Livrables

| Livrable | Description |
|----------|-------------|
| Analyse des flux | Audit des entrées/sorties sur les 90 derniers jours |
| Recommandations | Optimisation fiscale, structure juridique, épargne |
| Dashboard financier | Vue consolidée : CA, marges, prévisionnel 6 mois |
| Rapport PDF | Document remis au client avec les recommandations |
| Appel débrief | 1 call de 40 min pour présenter le rapport |

## Pour le client

- **Visibilité** — Comprendre où va l'argent et comment optimiser
- **Structure** — Passage de l'auto-entreprise à la SASU/EURL si pertinent
- **Trésorerie** — Conseils pour lisser les flux et éviter les trous
- **Investissement** — Orientations pour réinvestir les gains dans l'activité

## Déroulé

1. **J0** — Signature + paiement → Accès à un questionnaire financier
2. **J+3** — Analyse des données par l'équipe MARIN (ou partenaire comptable)
3. **J+7** — Call débrief 40 min + remise du rapport
4. **J+15** — Suivi optionnel (30 min) si le client a des questions

## Prérequis

- Le client doit fournir ses **3 derniers relevés bancaires pros** ou accès lecture à son outil compta
- Le client doit avoir **au moins 2 mois de ventes** via le système MARIN
- Le conseil financier est un **diagnostic**, pas un accompagnement continu

## Notes

- MARIN n'est pas un cabinet d'expertise comptable. Les recommandations sont des orientations, pas des actes juridiques/fiscaux.
- Si le client a besoin d'un vrai accompagnement comptable, MARIN peut recommander un expert-comptable partenaire.
- Offre valable uniquement pour les clients ayant terminé au moins 60 jours de parcours.
