# Upsell Agence 1 — Rachat d'Actifs

- **Prix :** 10 000 €
- **Cible :** Client ayant terminé le parcours 90 jours (ou en cours)

## Actifs rachetés

| Actif | Description | Valeur estimée |
|-------|-------------|----------------|
| Site e-commerce Shopify Pro | Store existant avec produits, thème, configurations | 12 000 € |
| Tunnel de vente | Funnel 22 steps Vite/React/Shadcn (actif) | 8 000 € |
| Contenu marketing | Scripts email, textes funnel, visuels | 3 000 € |
| Dashboard client | Instance dashboard existante | 2 000 € |
| Contrat de service | Client lié à MARIN (revenue récurrent) | Variable |
| Lead data | Base de leads nettoyés (niche du client) | Variable |

## Modèle économique

MARIN rachète les actifs pour les exploiter sur d'autres clients de la même niche :

- Le site + funnel servent de **template de duplication** pour les nouveaux clients
- Le contenu marketing est réutilisé (adapté) pour les campagnes futures
- Le dashboard devient un template dashboard
- Le contrat client existant reste chez MARIN (revenue récurrent)

## Pour le client

- **Liquidity event** — 10 000 € cash pour abandonner la structure
- **Arrêt des prélèvements** — Plus de mensualité MARIN
- **Non-compétence** — Le client s'engage à ne pas recréer de structure concurrente pendant 12 mois
- **Données** — Le client conserve l'accès à ses données clients (export CSV)

## Conditions

- Le site doit être **actif et fonctionnel** (pas un site vide)
- Le tunnel de vente doit avoir **au moins 1 mois de production**
- Le client doit **signer un accord de non-concurrence** (12 mois)
- Le rachat est effectué **après** la fin du contrat 90 jours (ou résiliation amiable)
- Paiement unique : **10 000 €** (virement ou chèque)

## Durée de validité

Offre valable pendant la durée du contrat + 30 jours après la fin.
