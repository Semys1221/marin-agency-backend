# Tone — Marin

Définit le ton et le style de communication pour toute l'agence. Applicable à l'écrit (emails, offres, contrats, landing pages) comme à l'oral (appels, coaching).

## Principes généraux

- **Professionnel** — langage soigné, pas d'argot, pas de familiarité.
- **Direct** — va à l'essentiel, pas de blabla, pas de remplissage.
- **Concis** — phrases courtes, paragraphes de 2-3 lignes max.
- **Valorisant** — le client est acteur de sa réussite, jamais infantilisé.

## À proscrire

| À éviter | Pourquoi |
|----------|----------|
| "On s'occupe de..." | Trop vague, sonne comme une nounou |
| "Pas d'inquiétude" | Minimise le problème du client |
| "Je comprends" | Abusé, préférer une reformulation précise |
| "Ça marche / ça roule" | Trop familier |
| "Du coup" | Tic de langage oral |
| "En tout cas" | Remplissage |
| Émojis (sauf onboarding) | Hors contexte, sauf messages de bienvenue |
| Points d'exclamation multiples | Un seul suffit, ou aucun |

## Style d'écriture

- **Tutoiement** : non. Toujours vouvoiement client.
- **Formules de politesse** : sobres. Pas de "Cordialement" systématique. Varier : "Bien cordialement", "Bien à vous", "Dans l'attente de vous lire".
- **Objets d'email** : commencer par un mot-clé utile, pas un teaser.
  - Correct : "Rappel : votre appel — Dupont Fils"
  - Incorrect : "Une question pour vous !"
- **Ton froid** : oui pour les emails transactionnels (Resend).
- **Ton chaleureux mais pro** : pour le nurturing (loop.so) et l'onboarding. Rester cadré, pas de familiarité.

## Correspondance outil / ton

| Outil | Ton |
|-------|-----|
| Instantly (cold) | Direct, professionnel, question ouverte |
| Resend (transactionnel) | Factuel, froid, informatif |
| loop.so (nurturing) | Chaleureux mais pro, rassurant sans être familier |
| Ticket system | Empathique mais efficace, solutions avant excuses |
| Contrats / offres | Juridique, précis, neutre |
| Coaching calls | Encouragant mais direct, orienté résultat |
